# Climaproof Model Selection Tool

## Install Model Selection Tool
### Linux
```shell
git clone 
cd climaproof-mst
docker build --rm -t climaproof/mst .
docker run -t -i -p 5100:5100 climaproof/mst
```

Open your Browser (e.g. Firefox) <http://127.0.0.1:5100>
